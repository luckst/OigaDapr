# OigaDapr

Project created using **Project Tye** and **Dapr**.

this uses redis to store data

## Run Locally

Clone the project

```bash
  Open a CMD
```

Go to the solution folder and run

```bash
  dotnet build
```

Run the project using tye

```bash
  tye run
```

Open the dashboard, usually:

```bash
  https://localhost:8000
```

Open the OigaWeb project
